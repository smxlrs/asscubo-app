Automatic Update

This widget can update its data automatically. You can set the following options:
- Update interval
- Days of week when the automatic update is enabled
- One or two time intervals when the automatic update is enabled

Timetable Span

You can choose if the widget will download the entire timetable of the day or for less time. For routes with many trains a day, you should set 6h or 3h.

Train Filter

Select which kinds of train to not show on the widget.

Custom Titles

Customize the widget title, replacing the automatically generated one.

Show trains till arrival

If set, the trains will be displayed on the widget till they arrive at destination.

All solutions

In some cases, it lets you see some economical solutions.
If this option is enabled, timetables download may be slower.

Show platform

In some cases it may not work properly.
If enabled, refreshing will be slower and it will consume more data.
