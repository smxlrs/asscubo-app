Station

You can choose a name from the suggestion or let the widget pick the nearest.

Widget title

You can set an optional title for the widget, in place of the station name.

Exclude trains

You can filter the displayed train types checking the corresponding boxes.

Automatic Update

This widget can update its data automatically. You can set the following options:
- Update interval
- Days of week when the automatic update is enabled
- One or two time intervals when the automatic update is enabled
