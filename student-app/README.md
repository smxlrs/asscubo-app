# Asscubo App (意语新生助手)

意语新生助手是一款基于 React Native (Expo) 开发的移动客户端，专为中国留学生提供在意大利生活和学习的各种便利工具。

## 功能特性

- **新生手册**：
  - 离线手册阅读与指南。
  - 支持阅读面板调整字体大小、行距。
  - 适配自动深色模式，并在会话中记住用户的底色设置（关闭 App 时重置）。
  - 支持直接无确认跳转到内部章节（支持 `2.2`、`#2.2` 或 `handbook://2.2` 等多种语法）。
- **意语离线词典**：
  - 支持多词典并发加载（MDX 格式，包括意汉、汉意、动词变位等）。
  - 支持输入前缀自动联想、模糊匹配及变位反查。
  - 支持中文反查（修复了 MDX 二分查找中由于 `localeCompare` 导致的中文词条查找失败问题）。
  - 释义采用 Shadow DOM 隔离渲染，防止 HTML 样式冲突，并在深色模式下自动调整不清晰的字体颜色。
- **汇率换算**：实时/离线汇率换算，支持手动一键更新。

## 项目结构

- `/app`：页面路由与业务逻辑（Expo Router）
- `/components`：通用 UI 组件
- `/context`：全局状态上下文（如主题设置）
- `/lib`：公共库（包括自定义的 MDX 解析器、数据库封装等）
- `/assets`：静态资源（图片、预置词典等）

## 开始使用

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npx expo start -c
```

### 3. 环境配置
- 新生手册数据托管在 Supabase，需在项目根目录创建 `.env` 文件并配置 Supabase URL 和 API Key。
- 预置词典 MDX 文件需放在 `assets/dicts/` 目录下。
